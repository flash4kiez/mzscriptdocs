# $ban
Банить пользователя.

## Использование
```py
$ban[(Guild ID);User ID;(Reason)]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Guild ID | Гильдия в которой нужно забанить пользователя. | Нет |
| User ID | Какого пользователя нужно забанить | Да |
| Reason | Причина бана. | Нет |
