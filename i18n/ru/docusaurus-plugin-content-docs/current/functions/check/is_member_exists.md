# $isMemberExists
Проверяет есть ли пользователь в гильдии.

## Использование
```py
$isMemberExists[(Guild ID);User ID]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Guild ID | Айди гильдии в которой должен быть человек. | Нет |
| User ID | Айди пользователя для проверки. | Да |
