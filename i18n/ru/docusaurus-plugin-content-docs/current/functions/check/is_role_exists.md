# $isRoleExists
Проверяет существует ли роль.

## Использование
```py
$isRoleExists[(Guild ID);Role ID]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Guild ID | Айди гильдии в которой должна быть роль. | Нет |
| Role ID | Айди роли которая будет проверяться. | Да |
