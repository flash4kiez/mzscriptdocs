# $isAdmin
Проверяет есть ли у пользователя права администратора.

## Использование
```py
$isAdmin[(Guild ID);User ID]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Guild ID | Айди гильдии где есть пользователь. | Нет |
| User ID | Айди пользователя для проверки. | Да |
