# $isChannelExists
Проверяет существует ли канал.

## Использование
```py
$isChannelExists[(Guild ID);Channel ID]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Guild ID | Айди гильдии где есть канал. | Нет |
| Channel ID | Айди канала для проверки. | Да |
