# $deleteCommand
Удаляет сообщения которое вызывает сообщения.

## Использование
```py
$deleteCommand[(Delay;Format)]
```

## Параметры
| Название | Описание | Обьязательно |
| -------- | -------- | ------------ |
| Delay | Через какое время будет удаленно сообщения? | Нет |
| Format | Формат времени (s, m, h). | Нет |
